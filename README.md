# 🔧 Shisela Nathi Lab

An AI-powered welding assistant that helps users:
- Estimate project cost 💰
- Calculate required materials 📏
- Generate step-by-step fabrication guides 🛠️
- Provide safety measures ⚠️
- Suggest alternative designs 🎨

---

## 🚀 Features

- Machine Learning cost prediction (Random Forest)
- AI-generated welding instructions
- Real-time web interface using Flask
- Interactive UI (HTML, CSS, JS)

---

## 🧠 Technologies Used

- Python (Flask)
- Scikit-learn
- Pandas
- OpenAI API
- HTML, CSS, JavaScript
